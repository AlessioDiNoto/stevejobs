Comandi document

Selezione degli elementi:

  • document.getElementById(id): Seleziona un elemento tramite l'ID.
  • document.getElementsByClassName(className): Seleziona elementi tramite la classe CSS.
  • document.getElementsByTagName(tagName): Seleziona elementi tramite il nome del tag HTML.
  • document.querySelector(selector): Seleziona il primo elemento che corrisponde al selettore CSS.
  • document.querySelectorAll(selector): Seleziona tutti gli elementi che corrispondono al selettore CSS.

Manipolazione del contenuto HTML:

  • document.createElement(tagName): Crea un nuovo elemento HTML con il tag specificato.
  • document.createTextNode(text): Crea un nuovo nodo di testo con il testo specificato.

Gestione degli eventi:

  • document.addEventListener(event, function): Aggiunge un gestore per l'evento specificato al documento.
  • document.removeEventListener(event, function): Rimuove un gestore di eventi precedentemente aggiunto.

Navigazione nella struttura del documento:

  • document.documentElement: Restituisce l'elemento radice (tipicamente <html>) del documento.
  • document.body: Restituisce l'elemento <body> del documento.

Altro:

  • document.title: Restituisce o imposta il titolo del documento.
  • document.URL: Restituisce l'URL del documento.
  • document.cookie: Restituisce o imposta i cookie associati al documento.

Oggetto Element

Nel DOM HTML, l'oggetto Element rappresenta un elemento HTML, come P, DIV, A, TABLE o qualsiasi altro elemento HTML.

Proprietà per la Manipolazione degli Attributi:

  • elemento.accessKey() - Imposta o restituisce l'attributo accesskey di un elemento
  • elemento.attributes - Restituisce un NamedNodeMap degli attributi di un elemento
  • elemento.className - Imposta o restituisce il valore dell'attributo class di un elemento
  • elemento.dir - Imposta o restituisce il valore dell'attributo dir di un elemento
  • elemento.id - Imposta o restituisce il valore dell'attributo id di un elemento
  • elemento.lang - Imposta o restituisce il valore dell'attributo lang di un elemento

Proprietà per la Manipolazione del Contenuto:

  • elemento.appendChild(newChild) - Aggiunge (appende) un nuovo nodo figlio a un elemento specificato
  • elemento.insertBefore() - Inserisce un nuovo nodo figlio prima di un nodo figlio esistente
  • elemento.removeChild() - Rimuove un nodo figlio da un elemento
  • elemento.replaceChild() - Sostituisce un nodo figlio in un elemento
  • elemento.innerHTML - Imposta o restituisce il contenuto di un elemento
  • elemento.innerText - Imposta o restituisce il contenuto testuale di un nodo e dei suoi discendenti
  • elemento.textContent - Imposta o restituisce il contenuto testuale di un nodo e dei suoi discendenti
  • elemento.insertAdjacentElement() - Inserisce un nuovo elemento HTML in una posizione relativa a un elemento
  • elemento.insertAdjacentHTML() - Inserisce un testo formattato HTML in una posizione relativa a un elemento
  • elemento.insertAdjacentText() - Inserisce testo in una posizione relativa a un elemento
  • elemento.outerHTML - Imposta o restituisce il contenuto di un elemento (inclusi tag di apertura e chiusura)
  • elemento.outerText - Imposta o restituisce il contenuto testuale esterno di un nodo e dei suoi discendenti
  • elemento.remove() - Rimuove un elemento dal DOM

Proprietà per la Navigazione nell'Albero DOM:

  • elemento.firstChild - Restituisce il primo nodo figlio di un elemento
  • elemento.firstElementChild - Restituisce il primo elemento figlio di un elemento
  • elemento.lastChild - Restituisce l'ultimo nodo figlio di un elemento
  • elemento.lastElementChild - Restituisce l'ultimo elemento figlio di un elemento
  • elemento.nextSibling - Restituisce il prossimo nodo nello stesso livello dell'albero dei nodi
  • elemento.nextElementSibling - Restituisce il prossimo elemento nello stesso livello dell'albero dei nodi
  • elemento.parentNode - Restituisce il nodo genitore di un elemento
  • elemento.parentElement - Restituisce l'elemento genitore di un elemento
  • elemento.previousSibling - Restituisce il nodo precedente nello stesso livello dell'albero dei nodi
  • elemento.previousElementSibling - Restituisce l'elemento precedente nello stesso livello dell'albero dei nodi
  • element.children: Restituisce una collezione degli elementi figlio di un elemento specificato.

Metodi per la Manipolazione degli Elementi:

  • elemento.addEventListener() - Allega un gestore di eventi a un elemento
  • elemento.removeEventListener() - Rimuove un gestore di eventi allegato con il metodo addEventListener()
  • elemento.getAttribute(attribute) - Restituisce il valore di un attributo di un elemento specificato
  • elemento.setAttribute(attribute, value) - Imposta o modifica il valore di un attributo specificato
  • elemento.removeAttribute(attribute) - Rimuove un attributo da un elemento specificato
  • elemento.getAttributeNode() - Restituisce un nodo attributo
  • elemento.setAttributeNode() - Imposta o modifica un nodo attributo
  • elemento.removeAttributeNode() - Rimuove un nodo attributo e restituisce il nodo rimosso
  • elemento.hasAttribute() - Restituisce true se un elemento ha un dato attributo
  • elemento.hasAttributes() - Restituisce true se un elemento ha attributi

Modifica degli attributi e degli stili degli elementi:

  • element.style.property = value: Imposta lo stile CSS di un elemento.

Altri Metodi:

  • elemento.contains() - Restituisce true se un nodo è un discendente di un nodo
  • elemento.isEqualNode() - Controlla se due elementi sono uguali
  • elemento.isSameNode() - Controlla se due elementi sono lo stesso nodo
  • elemento.matches() - Restituisce true se un elemento è corrisposto da un dato selettore CSS
  • elemento.normalize() - Unisce nodi testuali adiacenti e rimuove nodi testuali vuoti in un elemento
  • elemento.toString() - Converte un elemento in una stringa
